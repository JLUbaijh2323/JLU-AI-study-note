# 第一部分：核心重点

## I. 一元积分学：预处理、不等式与计算思想

**本章精神**：一元积分的难点不在于“算”，而在于“看”。即在计算前，如何通过“预处理”将“不可积”变为“可积”，将“复杂”变为“对称”。

### A. 积分的“预处理”战略

#### 1. 对称性与周期性的高级应用

基础的奇偶性应用在此不表。竞赛中的对称性核心是利用被积函数在积分区间 $[a,b]$ 上的 $f(x) = f(a+b-x)$ 不变性，或利用此变换消除“病态项”。

- 汤/蒲精华 (1) - 消除病态项：
    
    处理 $\int_{0}^{\pi} xf(\sin x)dx$。
    
    - 标准思路：令 $t = \pi - x$，则 $x = \pi - t$，$dx = -dt$。
        
        $I = \int_{\pi}^{0} (\pi - t) f(\sin(\pi - t)) (-dt) = \int_{0}^{\pi} (\pi - t) f(\sin t) dt$
        
        $I = \pi \int_{0}^{\pi} f(\sin t) dt - \int_{0}^{\pi} t f(\sin t) dt = \pi \int_{0}^{\pi} f(\sin t) dt - I$
        
    - **解得**：$2I = \pi \int_{0}^{\pi} f(\sin t) dt \implies I = \frac{\pi}{2} \int_{0}^{\pi} f(\sin x)dx$。
        
    - **思想提炼**：此变换的本质不是“记忆”，而是“中和”。在原积分中，$f(\sin x)$ 是关于 $x=\pi/2$ 对称的，而 $x$ 是“病态”的（不对称）。通过 $t = \pi - x$ 变换并与原式相加 ($2I$)，可以“中和”掉这个病态项 $x$，使其变为常数 $\pi$。
        
    - **决策链**：(1) 识别积分中的“不对称”因子 (如 $x$) 和“对称”因子 (如 $f(\sin x)$, $f(x^2)$)。(2) 寻找一个变换 $t = T(x)$ (如 $t=\pi-x$, $t=1/x$)，该变换能使对称因子不变，但能改变不对称因子。(3) 将原积分 $I$ 与变换后的积分 $I$ 相加 ($2I$) 或相减 ($0$)，以消除或简化。
        
- 汤/蒲精华 (2) - 华里士 (Wallis) 公式的推广：
    
    $\int_{0}^{\frac{\pi}{2}} \sin^n x dx = \int_{0}^{\frac{\pi}{2}} \cos^n x dx$ 的本质是 $\sin x \leftrightarrow \cos x$ 在 $[0, \pi/2]$ 上的对称性（通过 $t = \pi/2 - x$）。
    
    - **推广**：$\int_{0}^{\frac{\pi}{2}} f(\sin x, \cos x)dx$。如果 $f(\sin x, \cos x)$ 具有 $\sin \leftrightarrow \cos$ 的对称性，例如 $\int_{0}^{\frac{\pi}{2}} \frac{dx}{1+\tan^\alpha x} = \int_{0}^{\frac{\pi}{2}} \frac{\cos^\alpha x}{\cos^\alpha x + \sin^\alpha x} dx$。
        
    - 处理：$I = \int_{0}^{\frac{\pi}{2}} \frac{\cos^\alpha x}{\cos^\alpha x + \sin^\alpha x} dx$。
        
        令 $t = \pi/2 - x$，$I = \int_{0}^{\frac{\pi}{2}} \frac{\sin^\alpha t}{\sin^\alpha t + \cos^\alpha t} dt$。
        
        $2I = \int_{0}^{\frac{\pi}{2}} 1 dx = \pi/2 \implies I = \pi/4$。
        

#### 2. 特殊变换：(a) 倒代换 (b) 伽马/贝塔函数构造

- **(a) 倒代换 $t=1/x$** ：
    
    - **适用**：常用于 $[0, \infty]$ 上的积分，特别是当被积函数包含 $x$ 和 $1/x$ 的对称形式时。
        
    - **示例**：$I = \int_0^\infty \frac{\ln x}{1+x^2} dx$。
        
    - 处理：令 $t = 1/x$，$dx = -1/t^2 dt$。
        
        $I = \int_\infty^0 \frac{\ln(1/t)}{1+(1/t)^2} (-\frac{1}{t^2} dt) = \int_0^\infty \frac{-\ln t}{t^2+1} dt = -I$。
        
        $2I = 0 \implies I = 0$。
        
- **(b) 伽马/贝塔函数构造** ：
    
    - **核心**：$B(m,n) = \int_0^1 x^{m-1}(1-x)^{n-1}dx$ 和 $\Gamma(s) = \int_0^\infty x^{s-1}e^{-x}dx$。
        
    - **思想提炼**：不要死记 $\Gamma(1/2) = \sqrt{\pi}$。必须理解高斯积分 $\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}$ 是如何通过二重积分的极坐标变换得到的。这（ 提及的坐标系选择）是高斯积分的本质。
        
    - **竞赛级变形**：计算 $\int_{0}^{\infty} e^{-x^2 - a^2/x^2} dx$ ($a>0$)。
        
    - 处理：(1) 关键是配方 $x^2 + a^2/x^2 = (x - a/x)^2 + 2a$。(2) 尝试 $t = x - a/x$。 $dt = (1 + a/x^2) dx$。(3) 无法直接替换。(4) 蒲氏技巧：$I = \int_{0}^{\infty} e^{-(x^2 + a^2/x^2)} dx$。
        
        令 $t = a/x$，$I = \int_{\infty}^{0} e^{-(a^2/t^2 + t^2)} (-\frac{a}{t^2} dt) = \int_0^\infty \frac{a}{t^2} e^{-(t^2 + a^2/t^2)} dt$。
        
        $2I = \int_0^\infty (1 + \frac{a}{x^2}) e^{-(x^2 + a^2/x^2)} dx$。
        
        $2I = e^{-2a} \int_0^\infty (1 + \frac{a}{x^2}) e^{-(x - a/x)^2} dx$。
        
        令 $u = x - a/x$，$du = (1 + a/x^2) dx$。当 $x \to 0^+$，$u \to -\infty$。当 $x \to \infty$，$u \to \infty$。
        
        $2I = e^{-2a} \int_{-\infty}^{\infty} e^{-u^2} du = e^{-2a} \sqrt{\pi}$。
        
        $I = \frac{\sqrt{\pi}}{2} e^{-2a}$。
        

### B. 积分不等式证明的系统性构造

#### 1. 辅助函数构造法

- **基础**：证明 $\int_a^b f(x)dx > 0$。构造 $F(x) = \int_a^x f(t)dt$，利用 $F(a)=0$ 和 $F'(x)=f(x)$ 来分析 $F(b)$ 的符号。
    
- **拔高**：证明 $\int_a^b f(x)dx > \int_a^b g(x)dx$。构造 $H(x) = \int_a^x [f(t)-g(t)]dt$。
    
- **高级构造 (用于证明积分均值不等式)**：
    
    - **目标**：证明 $f(\frac{a+b}{2}) \le \frac{1}{b-a}\int_a^b f(x)dx$ (假设 $f$ 凸)。
        
    - **构造**：$F(t) = \int_a^{a+t} f(x)dx + \int_{b-t}^{b} f(x)dx$？不行。
        
    - **汤氏构造**：$F(t) = (b-a)\int_a^t f(x)dx - (t-a)\int_a^b f(x)dx$。
        

#### 2. 凹凸性 (Convexity) 与 詹森 (Jensen) 不等式

- **核心思想**：Jensen 不等式的积分形式是蒲河平课程中的一个核心思想。它提供了一个强大的“全局”放缩工具。
    
- **形式**：如果 $g(t)$ 是凸函数 ($g''(t) \ge 0$)，则 $g(\mathbb{E}[X]) \le \mathbb{E}[g(X)]$。
    
- **应用**：
    
    - (1) $\frac{1}{b-a}\int_a^b g(f(x))dx \ge g(\frac{1}{b-a}\int_a^b f(x)dx)$。
        
    - (2) 特别地，如果 $g(t)$ 是凸函数，$\frac{1}{b-a}\int_a^b g(x)dx \ge g(\frac{a+b}{2})$。
        
- **决策链**：(1) 识别题目中的函数 $g(t)$ 是否具有全局凹凸性 (如 $e^t$, $\ln t$, $t^p$)。(2) 识别“平均值”的形式 ($\frac{1}{b-a}\int_a^b \dots$)。(3) 应用 Jensen 不等式。
    
- **示例**：证明 $e^{\int_0^1 f(x)dx} \le \int_0^1 e^{f(x)}dx$ (假设 $\int_0^1 1 dx = 1$)。
    
- **处理**：(1) 设 $g(t) = e^t$。$g''(t) = e^t > 0$， $g$ 是凸函数。(2) 设 $X$ 是 $ 上的均匀分布，概率密度为 $p(x)=1$。(3) $\mathbb{E}[f(X)] = \int_0^1 f(x) \cdot 1 dx$。(4) $\mathbb{E}[g(f(X))] = \int_0^1 g(f(x)) \cdot 1 dx = \int_0^1 e^{f(x)}dx$。(5) $g(\mathbb{E}[f(X)]) = g(\int_0^1 f(x)dx) = e^{\int_0^1 f(x)dx}$。(6) 根据 Jensen，$g(\mathbb{E}[f(X)]) \le \mathbb{E}[g(f(X))]$，证毕。
    

#### 3. 柯西-施瓦茨 (Cauchy-Schwarz) 与 切比雪夫 (Chebyshev) 不等式

- **Cauchy**：$[\int_a^b f(x)g(x)dx]^2 \le \int_a^b f^2(x)dx \cdot \int_a^b g^2(x)dx$。
    
- Chebyshev：如果 $f, g$ 在 $[a,b]$ 上均单调同增（或同减），则
    
    $\frac{1}{b-a}\int_a^b f(x)g(x)dx \ge (\frac{1}{b-a}\int_a^b f(x)dx)(\frac{1}{b-a}\int_a^b g(x)dx)$。
    
- **思想提炼**：Chebyshev 的本质是“正相关”。两个同增函数“在 $x$ 大的地方都大，在 $x$ 小的地方都小”，因此它们的乘积的均值大于它们均值的乘积。如果 $f$ 增 $g$ 减，不等号反向。
    
- **决策**：识别被积函数中两个因子的“单调性”是使用此不等式的_唯一_前提。
    

### C. 变限积分与反常积分

#### 1. 反常积分收敛性的“比阶”精髓

这与级数 的“比阶”思想完全一致。

- **第一标尺 (P-积分)**：
    
    - (a) 瑕点 ($x \to 0^+$)：$\int_0^c \frac{dx}{x^p}$ ($p<1$ 收敛)。
        
    - (b) $\infty$ 点 ($x \to \infty$)：$\int_a^\infty \frac{dx}{x^p}$ ($p>1$ 收敛)。
        
- **第二标尺 (对数 P-积分)** ：
    
    - (a) $\infty$ 点 ($x \to \infty$)：$\int_a^\infty \frac{dx}{x(\ln x)^p}$ ($p>1$ 收敛)。
        
    - (b) 瑕点 ($x \to 0^+$)：$\int_0^c \frac{dx}{x(\ln x)^p}$ (无 p 要求，均收敛)。[易错点]
        
    - (c) 瑕点 ($x \to 1^+$)：$\int_1^c \frac{dx}{(x-1)^q (\ln x)^p}$。令 $t=x-1$，$\ln x = \ln(1+t) \sim t$。变为 $\int_0 \frac{dt}{t^q t^p} = \int_0 \frac{dt}{t^{p+q}}$。
        
- **核心思想**：$\int \frac{1}{x^p}$ 是“第一标尺”。$\int \frac{1}{x(\ln x)^p}$ 是“第二标尺”。当你用第一标尺比阶失败（例如 $\lim = 0$ 或 $\infty$），立即切换到第二标尺。
    
- **示例**：$\int_e^\infty \frac{dx}{x^p (\ln x)^q}$。
    
    - (1) 若 $p>1$， $q$ 任意。$\frac{1}{x^p (\ln x)^q} \sim \frac{1}{x^p}$ (取 $\epsilon$ 使 $p-\epsilon > 1$)。收敛。
        
    - (2) 若 $p<1$， $q$ 任意。$\frac{1}{x^p (\ln x)^q} \sim \frac{1}{x^p}$。发散。
        
    - (3) 若 $p=1$。变为 $\int_e^\infty \frac{dx}{x (\ln x)^q}$。此时 $q>1$ 收敛，$q \le 1$ 发散。
        

## II. 无穷级数：敛散性的本质与反例构造

**本章精神**：级数的灵魂在于“尾巴”。级数问题 的核心不是判别法本身，而是“通项的精细结构”以及“如何构造反例来打破常规判别法的前提条件”。

### A. 敛散性判别的核心思想

#### 1. “通项变小，收敛性扩大”：从绝对收敛到条件收敛

这是一个“本质思想”，是用户在查询中提到的。

- **收敛谱系**：$\sum |u_n|$ 收敛 (绝对收敛) $\implies \sum u_n$ 收敛 $\implies u_n \to 0$。
    
- **逆否命题**：$u_n \not\to 0 \implies \sum u_n$ 发散。
    
- **条件收敛**：$\sum u_n$ 收敛，但 $\sum |u_n|$ 发散（如 $\sum \frac{(-1)^n}{n}$）。
    
- **思想提炼**：
    
    - (1) 绝对收敛是“最强”的收敛，它不依赖于正负抵消。
        
    - (2) 条件收敛是“最弱”的收敛，它_完全_依赖于项的“正负抵消”。
        
    - (3) 黎曼重排定理：任意改变条件收敛级数的项的顺序，可以使其收敛于任意值，这说明它的“稳定性”极差。
        
    - (4) **应用 (判断性质题)**：任何“加强”通项的操作（如平方、取绝对值）都可能破坏条件收敛的“精巧平衡”。
        
        - Q: $\sum u_n$ 条件收敛 $\implies \sum u_n^2$ 收敛？
            
        - A: 否。反例：$u_n = \frac{(-1)^n}{\sqrt{n}}$。$\sum u_n$ 收敛 (Leibniz)。$\sum u_n^2 = \sum \frac{1}{n}$ 发散。
            
        - Q: $\sum u_n$ 绝对收敛 $\implies \sum u_n^2$ 收敛？
            
        - A: 是。$\sum |u_n|$ 收敛 $\implies |u_n| \to 0$。$\exists N, \forall n>N, |u_n|<1$。此时 $u_n^2 = |u_n|^2 < |u_n|$。由比较判别法，$\sum u_n^2$ 收敛。
            

#### 2. “比阶”的极限：对数 P-级数 (Logarithmic P-Series)

如 所述，$\sum \frac{1}{n^p}$ 是第一标尺。

- **第二标尺**：$\sum_{n=2}^\infty \frac{1}{n(\ln n)^p}$ ( $p>1$ 收敛)。
    
- **第三标尺**：$\sum_{n=3}^\infty \frac{1}{n \ln n (\ln \ln n)^p}$ ( $p>1$ 收敛)。
    
- **应用时机**：当使用第一标尺 $\sum \frac{1}{n^p}$ 的极限比较法失效时（极限为 0 或 $\infty$）。
    
- **汤/蒲精华**：在比阶时，不要只看 $n^p$。要看 $u_n$ 的“主部”。使用泰勒展开是看清主部的_唯一_系统性方法。
    
    - **示例**：判断 $\sum (\frac{1}{\sqrt{n}} - \sin(\frac{1}{\sqrt{n}}))$ 的敛散性。
        
    - **处理**：(1) 令 $x = 1/\sqrt{n}$。(2) $\sin x = x - \frac{x^3}{6} + o(x^3)$。(3) $x - \sin x = \frac{x^3}{6} + o(x^3)$。(4) $u_n = \frac{1}{6} (1/\sqrt{n})^3 + o(n^{-3/2}) = \frac{1}{6} n^{-3/2} + o(n^{-3/2})$。(5) $u_n \sim \frac{1}{6n^{3/2}}$。(6) $p=3/2 > 1$，收敛。
        

#### 3. 任意项级数的高级判别：Dirichlet 与 Abel

用于处理 $\sum u_n v_n$。

- **Abel 判别法**：
    
    - **条件**：(1) $\sum u_n$ 收敛 (2) $v_n$ 单调有界。
        
    - **本质**：为一个“收敛”级数乘上一个“稳定”的因子 $v_n$，收敛性不变。
        
    - **应用**：$\sum \frac{(-1)^n}{n}$ 收敛。$v_n = (1+1/n)$ 单调有界。故 $\sum \frac{(-1)^n}{n} (1+1/n)$ 收敛。
        
- **Dirichlet 判别法**：
    
    - **条件**：(1) $\sum u_n$ 部分和 $U_N = \sum_{n=1}^N u_n$ 有界 (2) $v_n$ 单调 $\to 0$。
        
    - **本质 (精华)**：这是_真正_的精华。它将 $\sum u_n$ 的“振荡性”（有界，但不必收敛）与 $v_n$ 的“衰减性”（单调 $\to 0$）结合起来，_创造_了收敛。
        
    - 决策链 ：何时使用 Dirichlet？
        
        - (1) 看到 $\sum \frac{f(n)}{g(n)}$。
            
        - (2) $g(n)$ 是“平滑”的，单调 $\to \infty$ (如 $n, \ln n$)。
            
        - (3) $f(n)$ 是“振荡”的 (如 $(-1)^n, \sin n, \cos n$)。
            
        - (4) 令 $u_n = f(n)$, $v_n = 1/g(n)$。
            
        - (5) **关键**：证明 $U_N = \sum_{n=1}^N f(n)$ 有界。
            
            - $f(n) = (-1)^n \implies |U_N| \le 1$。
                
            - $f(n) = \sin(nx)$ (只要 $x \ne 2k\pi$)：$\sum_{n=1}^N \sin(nx) = \frac{\cos(x/2) - \cos((N+1/2)x)}{2\sin(x/2)}$。$|U_N| \le \frac{2}{|2\sin(x/2)|}$ (有界)。
                
    - **示例**：$\sum_{n=1}^\infty \frac{\sin n}{n}$。(1) $u_n = \sin n$，$U_N$ 有界。(2) $v_n = 1/n$ 单调 $\to 0$。(3) 由 Dirichlet，收敛。
        

### B. 高难反例的构造原则

提到了构造反例。这是蒲河平课程的重点。构造反例的本质是“在规则的边缘试探”。找到判别法 (Leibniz, Dirichlet) 的_每一个_前提条件（如“单调性”），然后构造一个_仅_破坏该条件的函数。

1. **破坏“单调性” (Leibniz)**：Leibniz 判别法 $\sum (-1)^n u_n$ 要求 $u_n$ _单调_ $\to 0$。
    
    - 蒲氏经典反例 ：构造一个 $u_n \to 0$ 但不单调的 $u_n$，使得级数发散。
        
    - $u_n = \begin{cases} \frac{1}{n}, & n \text{ is odd} \\ \frac{1}{n^2}, & n \text{ is even} \end{cases}$
        
    - $u_n \to 0$ 成立。
        
    - $\sum (-1)^n u_n = (\frac{1}{1} - \frac{1}{2^2}) + (\frac{1}{3} - \frac{1}{4^2}) + \dots + (\frac{1}{2k-1} - \frac{1}{(2k)^2}) + \dots$
        
    - 通项 $a_k = \frac{1}{2k-1} - \frac{1}{(2k)^2} = \frac{4k^2 - 2k + 1}{(2k-1)(2k)^2} \sim \frac{4k^2}{2k \cdot 4k^2} = \frac{1}{2k}$。
        
    - $\sum a_k$ 与 $\sum \frac{1}{2k}$ 同敛散，发散。
        
2. 构造绝对收敛 vs 条件收敛 ：
    
    - **问题**：若 $\sum u_n, \sum v_n$ 均条件收敛，$\sum u_n v_n$ 呢？
        
    - **反例 1 (收敛)**：$u_n = v_n = \frac{(-1)^n}{n}$。$\sum u_n, \sum v_n$ 均条件收敛。$\sum u_n v_n = \sum \frac{1}{n^2}$ 绝对收敛。
        
    - **反例 2 (发散)**：$u_n = v_n = \frac{(-1)^n}{\sqrt{n}}$。$\sum u_n, \sum v_n$ 均条件收敛。$\sum u_n v_n = \sum \frac{1}{n}$ 发散。
        
    - **结论**：不一定。
        

### C. 幂级数与和函数求解

提到了“非标准和函数”。

- **标准**：$S(x) = \sum a_n x^n$。通过逐项积分、逐项求导、错位相减来求解。
    
- **非标准策略 1：构造微分方程**：
    
    - **时机**：如果 $S(x)$ 难以直接求和，但 $S'(x)$ 和 $S(x)$ 之间存在简单关系。
        
    - **示例**：求 $\sum_{n=0}^\infty \frac{n!}{(2n+1)!} x^n$ (注：此例过于复杂，仅为示意)。求 $S(x) = \sum_{n=0}^\infty \frac{x^n}{n!}$。
        
    - **决策链**：(1) $S'(x) = \sum_{n=1}^\infty \frac{n x^{n-1}}{n!} = \sum_{n=1}^\infty \frac{x^{n-1}}{(n-1)!} = \sum_{k=0}^\infty \frac{x^k}{k!} = S(x)$。(2) 建立 DE：$S'(x) = S(x)$。(3) 求解 $S(x) = C e^x$。(4) 用 $S(0) = a_0 = \frac{1}{0!} = 1$ 定常数 $C=1$。(5) $S(x) = e^x$。
        
- **非标准策略 2：泰勒展开逆用**：
    
    - **时机**：要求 $\sum a_n$ (即 $S(1)$)，但 $S(x)$ 无法用标准方法求和。
        
    - **思路**：(1) 尝试_猜_ $S(x)$ 对应的函数 $f(x)$。(2) 证明 $a_n = \frac{f^{(n)}(0)}{n!}$。(3) 则 $\sum a_n = f(1)$。

## III. 中值定理：辅助函数的“神圣”构造法

**本章精神**：中值定理 的所有题目都是“逆向工程”。题目给出了 $f(\xi)$ 的某个复杂表达式 $G(\xi)=0$，你需要找到它的“原函数” $F(x)$，使得 $F(x)$ 满足罗尔定理的条件（如 $F(a) = F(b)$ 或 $F(a)=F'(a)=0$ 等），然后应用罗尔定理（或泰勒）得到 $F'(\xi)=G(\xi)=0$ 或更高阶形式。

### A. 辅助函数构造的系统性模式

不要“猜” $F(x)$。要“解” $F(x)$。

- **决策链**：(1) 将欲证的 $\xi$ 表达式 $G(\xi) = 0$ 中所有的 $\xi$ 换成 $x$。(2) 得到 $G(x) = 0$。(3) 观察 $G(x)$ 的结构，看它是否像某个函数 $F(x)$ 的导数 $F'(x)$。(4) 如果是，构造 $F(x)$ 并应用罗尔定理。(5) 如果不是，尝试“还原”。
    

以下是为响应 需求而设计的系统性决策表：

**表格 3：中值定理辅助函数构造模式**

|(6) $\frac{f(b)-f(a)}{g(b)-g(a)} = \frac{f'(\xi)}{g'(\xi)}$|$F(x) = [f(b)-f(a)]g(x) - [g(b)-g(a)]f(x)$|还原柯西中值定理的证明过程。|

### B. 罗尔定理的深度应用：零点个数分析

- **基础**：证明 $f'(x) = k$ 有根。构造 $F(x) = f(x) - kx$。
    
- 高阶零点分析 ：证明 $f^{(n)}(\xi) = 0$ 存在。
    
    - **标准**：需要找到 $n+1$ 个点 $x_0, x_1, \dots, x_n$ 使得 $f(x)$ 在这些点上取值相等（通常为 0）。
        
    - **决策链**：(1) $f(x)$ 在 $x_0, \dots, x_n$ 处 $n+1$ 个零点。(2) (罗尔) $\implies f'(x)$ 在 $n$ 个零点。(3) (罗尔) $\implies f''(x)$ 在 $n-1$ 个零点... (n次罗尔) $\implies f^{(n)}(x)$ 在 1 个零点 $\xi$。
        
- 汤/蒲精华 ：真正的难点是 $f(x)$ 没有 $n+1$ 个零点，而是“在一点上提供了高阶信息”。
    
    - **示例**：$f(a)=f(b)=0$，且 $f(a)=f'(a)=0$ (即 $a$ 是 $f(x)$ 的二重根)。
        
    - **分析**：(1) $f(a)=0, f(b)=0 \implies \exists \xi_1 \in (a,b)$ s.t. $f'(\xi_1)=0$。(2) 我们已知 $f'(a)=0$。(3) 现在 $f'(x)$ 在 $a$ 和 $\xi_1$ 两个点上为 0。(4) (对 $f'(x)$ 应用罗尔) $\implies \exists \xi \in (a, \xi_1) \subset (a,b)$ s.t. $(f')'(\xi) = f''(\xi) = 0$。
        
    - **结论**：$f(a)=f(b)=0$ 加上 $f'(a)=0$，这“三个”信息 $\implies \exists \xi$ s.t. $f''(\xi)=0$。
        

### C. 泰勒中值定理的精华

泰勒公式在中值定理中的应用主要是指拉格朗日余项。

- **佩亚诺 (Peano) 余项**：$f(x) = \sum_{k=0}^n \frac{f^{(k)}(x_0)}{k!} (x-x_0)^k + o((x-x_0)^n)$。
    
    - **本质**：在 $x_0$ 点的“局部最佳多项式近似”。
        
    - 用途 ：_唯一_用途是求极限。
        
- **拉格朗日 (Lagrange) 余项**：$R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!} (x-x_0)^{n+1}$ ( $\xi$ 在 $x_0, x$ 之间)。
    
    - **本质**：“全局”估计，是中值定理的推广（当 $n=0$ 时， $f(x) = f(x_0) + f'(\xi)(x-x_0)$，即拉格朗日中值定理）。
        
    - **用途**：(a) 证明不等式 (b) 估计近似误差 (c) 证明高阶导数存在性 。
        
- **泰勒在求导中的“降维打击”** ：
    
    - **问题**：求 $f^{(n)}(x_0)$。
        
    - **决策**：当直接求导（用莱布尼兹） 非常困难 (e.g., $f(x) = \frac{\sin x}{x}$ 在 $x=0$) 时，应立即转向泰勒展开 (Peano)。
        
    - **决策链**：(1) 题目要求 $f^{(n)}(0)$。(2) 将 $f(x)$ 在 $x=0$ 处展开：$f(x) = \frac{1}{x} (x - \frac{x^3}{3!} + \frac{x^5}{5!} - \dots) = 1 - \frac{x^2}{3!} + \frac{x^4}{5!} - \dots$。(3) $f(x)$ 的泰勒展开是唯一的。(4) 比较 $f(x) = \sum_{n=0}^\infty a_n x^n$。(5) $a_n = \frac{f^{(n)}(0)}{n!}$。(6) 因此 $f^{(n)}(0) = n! \cdot a_n$。(7) 示例：$f^{(4)}(0) = 4! \cdot a_4 = 4! \cdot (\frac{1}{5!}) = \frac{1}{5}$。$f^{(3)}(0) = 3! \cdot a_3 = 3! \cdot 0 = 0$。

## IV. 极限：非泰勒方法的精髓

**本章精神**：泰勒展开 是求极限的“核武器”，但不是万能的。当函数不可展（或在 $\infty$ 处）或为数列时，需要 和 提到的“非泰勒方法”。

### A. 斯托尔兹-切萨罗 (Stolz-Cesàro) 定理

- **形式** ($\frac{\infty}{\infty}$ 型)：$\lim \frac{a_n}{b_n}$ ( $b_n \to \infty$ 单调)。若 $\lim \frac{a_{n+1}-a_n}{b_{n+1}-b_n} = L$，则 $\lim \frac{a_n}{b_n} = L$。
    
- **本质**：Stolz 是离散版的 L'Hôpital。
    
- **应用 1：处理 $a_n$ 的复杂性 (降维)**：
    
    - **示例**：$\lim_{n \to \infty} \frac{1!+2!+...+n!}{n!}$。
        
    - 处理：$a_n = \sum_{k=1}^n k!$, $b_n = n!$。$b_n$ 单增 $\to \infty$。
        
        $\lim \frac{a_{n+1}-a_n}{b_{n+1}-b_n} = \lim \frac{(n+1)!}{(n+1)! - n!} = \lim \frac{(n+1)!}{n!(n+1-1)} = \lim \frac{n+1}{n} = 1$。
        
    - 原极限 $= 1$。
        
- **应用 2：处理 $\frac{0}{0}$ 型 / 递归数列**：
    
    - **问题**：$\lim_{n \to \infty} n (x_n - a)$，其中 $x_{n+1} = f(x_n)$ 且 $x_n \to a$。
        
    - **分析**：这是 $\infty \cdot 0$ 型。
        
    - 标准处理 (Stolz)：写成 $\frac{n}{1/(x_n-a)}$ ( $\infty/\infty$ 型)。
        
        $\lim \frac{(n+1)-n}{1/(x_{n+1}-a) - 1/(x_n-a)} = \lim \frac{1}{\frac{x_n - x_{n+1}}{(x_{n+1}-a)(x_n-a)}}$。
        
    - 蒲氏思想 ：$\lim (x_n - a) = 0$。
        
        $x_{n+1}-a = f(x_n) - f(a)$。
        
        (泰勒) $f(x_n) - f(a) = f'(a)(x_n-a) + \frac{f''(\xi)}{2}(x_n-a)^2 \approx f'(a)(x_n-a)$。
        
    - 这启发我们 $\lim \frac{x_{n+1}-a}{x_n-a} = f'(a)$。
        
    - 高级处理：$\lim n(x_n-a)$。
        
        设 $y_n = x_n - a$。$\lim \frac{n}{1/y_n}$。
        
        $\lim \frac{1}{1/y_{n+1} - 1/y_n} = \lim \frac{y_n y_{n+1}}{y_n - y_{n+1}} = \lim \frac{y_n y_{n+1}}{y_n - f(y_n+a) + a}$。
        
        $f(y_n+a) = f(a) + f'(a)y_n + \frac{f''(a)}{2}y_n^2 + o(y_n^2)$。
        
        $f(y_n+a) - a = f'(a)y_n + \frac{f''(a)}{2}y_n^2 + o(y_n^2)$。
        
        $\lim \frac{y_n \cdot (f'(a)y_n)}{y_n - [f'(a)y_n + \frac{f''(a)}{2}y_n^2]} = \lim \frac{f'(a)y_n^2}{y_n(1-f'(a)) - \frac{f''(a)}{2}y_n^2}$。
        
    - 如果 $f'(a) \ne 1$， $\lim = 0$。
        
    - 如果 $f'(a) = 1$， $\lim \frac{y_n^2}{- \frac{f''(a)}{2}y_n^2} = -\frac{2}{f''(a)}$。
        
    - **结论**：Stolz 在此处的应用极其依赖于泰勒展开的深度。
        

### B. 积分定义求极限

- **形式**：$\lim_{n \to \infty} \frac{b-a}{n} \sum_{i=1}^n f(a + i \frac{b-a}{n}) = \int_a^b f(x)dx$。
    
- **难点 1：识别 $\frac{i}{n}$**。
    
- **难点 2：处理 $\ln$ 与乘积 ( $\lim \sqrt[n]{P_n}$ )**：
    
    - **示例**：$\lim_{n \to \infty} \frac{1}{n} \sqrt[n]{(n+1)(n+2)...(n+n)}$。
        
    - **处理**：(1) 设 $A_n = \dots$。(2) 取 $\ln$ 变为和式：$\ln A_n = \frac{1}{n} \ln (\frac{(2n)!}{n!}) - \ln n$ (错误)。
        
    - 正确处理：
        
        $A_n = \sqrt[n]{\frac{(n+1)}{n} \cdot \frac{(n+2)}{n} \dots \frac{(n+n)}{n}} = \sqrt[n]{\prod_{i=1}^n (1 + i/n)}$。
        
        (3) $\ln A_n = \frac{1}{n} \sum_{i=1}^n \ln(1+i/n)$。
        
        (4) 这是黎曼和：$[a,b] = $，$f(x) = \ln(1+x)$。
        
        (5) $\lim \ln A_n = \int_0^1 \ln(1+x)dx = [ (x+1)\ln(x+1) - x ] \Big|_0^1 = 2\ln 2 - 1$。
        
        (6) 原极限 $\lim A_n = e^{2\ln 2 - 1} = e^{\ln 4 - 1} = 4/e$。
        

### C. 微分方程求极限

- **基础**：若 $\lim_{x \to \infty} f'(x) = L > 0$，则 (由 L'Hôpital) $\lim_{x \to \infty} \frac{f(x)}{x} = L$，故 $\lim_{x \to \infty} f(x) = \infty$。
    
- 蒲氏经典 ：若 $\lim_{x \to \infty} (f'(x) + a f(x)) = L$ ($a>0$)，则 $\lim_{x \to \infty} f(x) = L/a$。
    
- **思想提炼**：
    
    - (1) 设 $g(x) = f'(x) + a f(x)$，则 $\lim_{x \to \infty} g(x) = L$。
        
    - (2) 这是一个一阶线性 DE $f' + af = g(x)$。
        
    - (3) 解得 $f(x) = e^{-ax} (\int e^{at} g(t) dt + C)$。
        
    - (4) 求 $\lim_{x \to \infty} f(x) = \lim_{x \to \infty} \frac{\int e^{at} g(t) dt + C}{e^{ax}}$ ($\frac{\infty}{\infty}$ 型)。
        
    - (5) (L'Hôpital) $\lim_{x \to \infty} \frac{e^{ax} g(x)}{a e^{ax}} = \lim_{x \to \infty} \frac{g(x)}{a} = \frac{L}{a}$。
        
    - (6) 证毕。此题的本质是“解微分方程”+“L'Hôpital”。
        

## V. 微分方程：非标准模型的求解策略

**本章精神**：考研大纲内的 DE (可分离变量、一阶线性、伯努利、齐次、二阶常系数) 都是基础。竞赛的难点在于 和 提到的“非标准”方程，它们需要特殊的“变量代换”或“几何洞察”。

### A. 特殊方程的识别与降阶

#### 1. 克莱罗 (Clairaut) 方程

- **形式**：$y = xy' + f(y')$。
    
- **汤/蒲精华**：
    
    - (1) 设 $p=y'$。$y = xp + f(p)$。
        
    - (2) 对 $x$ 求导：$y' = p + xp' + f'(p)p'$。
        
    - (3) $p = p + xp' + f'(p)p' \implies p'[x + f'(p)] = 0$。
        
    - (4) **解 1 (通解)**：$p' = 0 \implies p=C \implies y = Cx + f(C)$ (直线族)。
        
    - (5) **解 2 (奇解/包络)**：$x + f'(p) = 0 \implies x = -f'(p)$。代回原式 $y = -f'(p)p + f(p)$。这是奇解的参数方程。
        

#### 2. 拉格朗日 (Lagrange) 方程

- **形式**：$y = xg(y') + f(y')$。
    
- **汤/蒲精华**：
    
    - (1) 设 $p=y'$。$y = xg(p) + f(p)$。
        
    - (2) 对 $x$ 求导：$p = g(p) + xg'(p)p' + f'(p)p'$。
        
    - (3) $p - g(p) = [xg'(p) + f'(p)] \frac{dp}{dx}$。
        
    - (4) $\frac{dx}{dp} = \frac{xg'(p)}{p-g(p)} + \frac{f'(p)}{p-g(p)}$。
        
    - (5) $\frac{dx}{dp} - \frac{g'(p)}{p-g(p)} x = \frac{f'(p)}{p-g(p)}$。
        
    - (6) 这变成了 $x$ 关于 $p$ 的一阶线性 DE。先解出 $x=x(p)$，再连同 $y = x(p)g(p) + f(p)$ 得到参数方程。
        

#### 3. 对称方程组

- **形式**：$\frac{dx}{P(x,y,z)} = \frac{dy}{Q(x,y,z)} = \frac{dz}{R(x,y,z)}$。
    
- **汤/蒲精华**：核心是“合比定理”。
    
    - (1) 寻找 $\lambda, \mu, \nu$ (可以是 $x,y,z$ 的函数) 使得 $\lambda P + \mu Q + \nu R = 0$。
        
    - (2) 则 $\lambda dx + \mu dy + \nu dz = 0$。
        
    - (3) 如果 $\lambda dx + \mu dy + \nu dz = d(F_1(x,y,z))$ (是某个 $F_1$ 的全微分)。
        
    - (4) 积分得到第一个首次积分 $F_1(x,y,z) = C_1$。
        
    - (5) 重复此过程找到 $F_2(x,y,z) = C_2$。
        
    - **示例**：$\frac{dx}{y-z} = \frac{dy}{z-x} = \frac{dz}{x-y}$。
        
    - (1) $\lambda=1, \mu=1, \nu=1$。$P+Q+R = (y-z)+(z-x)+(x-y)=0$。
        
    - (2) $1 dx + 1 dy + 1 dz = 0 \implies d(x+y+z) = 0 \implies F_1 = x+y+z = C_1$。
        
    - (3) $\lambda=x, \mu=y, \nu=z$。$xP+yQ+zR = x(y-z)+y(z-x)+z(x-y)=0$。
        
    - (4) $x dx + y dy + z dz = 0 \implies d(\frac{1}{2}x^2 + \frac{1}{2}y^2 + \frac{1}{2}z^2) = 0 \implies F_2 = x^2+y^2+z^2 = C_2$。
        

## VI. 线面积分：计算规范与“补面”思想

**本章精神**：用户提到“总是算错数”。 强调“精华”和“补面”。这说明计算的难点不在于积分本身，而在于“建系”、“定号”和“公式选择”。

### A. 复杂计算的“防错”规范

#### 1. 第一类 (对弧长/面积) vs 第二类 (对坐标)

- **本质**：第一类是标量（$ds, dS$），与方向无关，恒为正。第二类是矢量（$dx, dy dz$），与方向（法向量、路径方向）_强相关_。
    
- **防错**：永远在第二类积分的右上角标出**方向** ( $\int_{\Gamma^+} \dots$ 或 $\iint_{\Sigma^+} \dots$ )。
    

#### 2. 规范化步骤 (以第二类曲面积分 $\iint_\Sigma P dy dz + Q dx dz + R dx dy$ 为例)

- **(1) 检查 $\Sigma$ 是否封闭**。
    
    - 若封闭，立即考虑**高斯 (Gauss) 公式** 。$\iint_{\Sigma_{\text{外}}} = \iiint_V (\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}) dV$。
        
- **(2) 若不封闭，检查边界线 $L$**。
    
    - 若 $L$ 简单（如平面圆），立即考虑**斯托克斯 (Stokes) 公式** 。$\iint_\Sigma (\nabla \times \vec{F}) \cdot d\vec{S} = \int_L \vec{F} \cdot d\vec{l}$。(注意：斯托克斯只能用于旋度场)。
        
- **(3) 如果 (1) 和 (2) 都不可行**（例如 $\vec{F}$ 不是旋度场，或高斯后的 $\iiint \nabla \cdot \vec{F} dV$ 更复杂），退守“参数化投影”。
    
- **(4) 投影法**：
    
    - (a) 确定投影面 (e.g., $xy$ 面)。
        
    - (b) 确定 $\Sigma$ 的方程 $z=z(x,y)$。
        
    - (c) 计算：$dy dz = \mp \frac{\partial z}{\partial x} dx dy$，$dx dz = \mp \frac{\partial z}{\partial y} dx dy$，$dx dy = \pm dx dy$。
        
    - (d) **定号**：这是 90% 的错误来源。
        
        - $dx dy$：看 $\Sigma$ 的法向量 $\vec{n}$ 的 $z$ 分量 $\vec{n} \cdot \vec{k}$ 是 > 0 还是 < 0。> 0 (上侧) 取正，< 0 (下侧) 取负。
            
        - $dy dz$：看 $\vec{n} \cdot \vec{i}$。
            
        - $dx dz$：看 $\vec{n} \cdot \vec{j}$。
            

#### 3. 对称性

- **蒲氏思想**：(a) 积分区域对称，(b) 被积函数“奇偶性”匹配。
    
- **第一类积分**：对称性分析_安全_。$\iint_D x dS$。若 $D$ 关于 $yz$ 面对称，$x$ 是奇函数，积分为 0。
    
- **第二类积分**：对称性分析_极其危险_。
    
    - **示例**：$\iint_\Sigma x dy dz$。 $\Sigma$ 为球面 $x^2+y^2+z^2=R^2$，取外侧。
        
    - **错误分析**：$\Sigma$ 关于 $yz$ 面 ($x=0$) 对称。被积函数 $P=x$ 是 $x$ 的奇函数。所以积分 $= 0$。
        
    - **正确分析 (高斯)**：(1) $\vec{F} = (x, 0, 0)$。(2) $\Sigma$ 封闭。(3) (高斯) $\iint_\Sigma = \iiint_V \nabla \cdot \vec{F} dV = \iiint_V 1 dV = \frac{4}{3}\pi R^3 \ne 0$。
        
    - **思想提炼**：为什么对称性分析失败了？因为 $dy dz$ 不是一个标量 $dS$。$dy dz = (\vec{n} \cdot \vec{i}) dS$。
        
        - (1) 在 $\Sigma$ 的右半球 ($x>0$)， $x$ 为正，法向量 $\vec{n}$ 的 $x$ 分量 $(\vec{n} \cdot \vec{i})$ 也为正。
            
        - (2) 在 $\Sigma$ 的左半球 ($x<0$)， $x$ 为负，法向量 $\vec{n}$ 的 $x$ 分量 $(\vec{n} \cdot \vec{i})$ 也为负。
            
        - (3) 被积“元” $x dy dz \approx x \cdot (\vec{n} \cdot \vec{i}) dS$ 是 (正 $\times$ 正) 或 (负 $\times$ 负)，_恒为正_！
            
        - (4) 奇 $\times$ 奇 = 偶。积分不为 0。
            

### B. 高斯 (Gauss) 与 斯托克斯 (Stokes) 的本质

#### 1. “补面” (Fill-a-Surface)

这是 和 的核心。用于计算“非封闭曲面” $\Sigma$ 上的 $\iint_\Sigma \vec{F} \cdot d\vec{S}$ (Gauss) 或 $\iint_\Sigma \nabla \times \vec{F} \cdot d\vec{S}$ (Stokes)。

- **Gauss 补面**：
    
    - **决策链**：(1) 求 $\iint_\Sigma \vec{F} \cdot d\vec{S}$ ($\Sigma$ 不封闭)。(2) 构造一个（或多个）简单的“盖子” $\Sigma_1$ (如 $z=0$ 的平面)，使 $\Sigma + \Sigma_1$ 构成一个封闭曲面 $\Sigma_{total}$（指向外侧）。(3) $\iint_{\Sigma_{total}} \vec{F} \cdot d\vec{S} = \iiint_V \nabla \cdot \vec{F} dV$。(4) $\iint_{\Sigma} + \iint_{\Sigma_1} = \iiint_V$。(5) **答案**： $\iint_{\Sigma} = \iiint_V \nabla \cdot \vec{F} dV - \iint_{\Sigma_1} \vec{F} \cdot d\vec{S}$。(6) **前提**：$\iiint_V$ (体积分) 和 $\iint_{\Sigma_1}$ (简单平面积分) 必须比原积分 $\iint_\Sigma$ 容易。
        
    - **示例**：$\Sigma$ 是 $z=\sqrt{x^2+y^2}$ ($z \le 1$)，取上侧。$\vec{F} = (x,y,z)$。
        
    - **处理**：(1) $\Sigma$ 不封闭。(2) 补一个“盖子” $\Sigma_1$：$z=1, x^2+y^2 \le 1$，法向量 $\vec{n}=(0,0,1)$ (为了构成外侧)。(3) $\Sigma_{total} = \Sigma + \Sigma_1$。(4) $\iiint_V \nabla \cdot \vec{F} dV = \iiint_V 3 dV = 3 \cdot \frac{1}{3}\pi (1)^2 (1) = \pi$。(5) $\iint_{\Sigma_1} \vec{F} \cdot d\vec{S} = \iint_{x^2+y^2 \le 1} (x,y,1) \cdot (0,0,1) dx dy = \iint_{x^2+y^2 \le 1} 1 dx dy = \pi$。(6) $\iint_\Sigma + \iint_{\Sigma_1} = \iiint_V \implies \iint_\Sigma + \pi = \pi \implies \iint_\Sigma = 0$。
        
- **Stokes 补面 (逆用)**：
    
    - **标准**：$\int_L \vec{F} \cdot d\vec{l} = \iint_\Sigma (\nabla \times \vec{F}) \cdot d\vec{S}$。我们选择_最简单_的 $\Sigma$ (如平面) 来计算 $\int_L$。
        
    - 汤/蒲精华 ：$\int_L$ 的计算难度 $\ll \iint_\Sigma$ 的计算难度。所以 Stokes 定理的_主要_用途是“反向”使用：
        
    - **问题**：计算 $\iint_\Sigma (\nabla \times \vec{F}) \cdot d\vec{S}$，其中 $\Sigma$ 是一个复杂曲面。
        
    - **决策链 (Stokes 逆用)**：(1) 求 $\iint_\Sigma (\nabla \times \vec{F}) \cdot d\vec{S}$。(2) 识别 $\Sigma$ 的边界 $L$。(3) **答案**：$\int_L \vec{F} \cdot d\vec{l}$。(4) **示例**：$\Sigma$ 是 $z = x^2+y^2$ 上 $z \le 4$ 的部分。求 $\iint_\Sigma (\nabla \times \vec{F}) \cdot d\vec{S}$。
        
    - **处理**：(1) $\Sigma$ 是一个复杂的抛物面。(2) 它的边界 $L$ 是 $z=4, x^2+y^2=4$ 的一个圆。(3) $\iint_\Sigma = \int_L \vec{F} \cdot d\vec{l}$。计算 $\int_L$ 远比计算 $\iint_\Sigma$ 简单。

## VII. 导数与微分：高阶导数的决策框架

**本章精神**： 和 提出了一个“决策框架”。求 $f^{(n)}(x_0)$ 时，你选择什么武器？

### A. 决策表：何时使用

|**问题特征 (Problem Characteristic)**|**推荐方法 (Recommended Method)**|**汤/蒲精华 (Key Insight/Pitfall)**|

|(1) $f(x) = g(x)h(x)$。$g,h$ 的高阶导数易求 (e.g., $x^k, e^{ax}, \sin(ax)$)|**莱布尼兹 (Leibniz)** $ (uv)^{(n)} = \sum C_n^k u^{(n-k)} v^{(k)} $|$f(x) = x^k \sin(ax)$。$v=x^k$ 在 $k+1$ 次求导后为 0，Leibniz 展开是有限项。|
|(2) 求 $f^{(n)}(x_0)$ (在某一点)，且 $f(x)$ 在 $x_0$ 处的泰勒展开易求|**泰勒展开 (Taylor Expansion)**|(见 III.C) $f(x) = \frac{\ln(1+x)}{x}$。$f(x) = \sum a_k x^k$。$f^{(n)}(0) = n! \cdot a_n$。这是“降维打击”。|

### B. 隐函数与参数方程的高阶导

- **汤/蒲精华**：
    
    1. **隐函数 $F(x,y)=0$**：
        
        - (1) $F'_x + F'_y y' = 0 \implies y' = -F'_x / F'_y$。
            
        - (2) 再次对 $x$ 求导：$F''_{xx} + F''_{xy} y' + (F''_{yx} + F''_{yy} y') y' + F'_y y'' = 0$。(注意 $y'$ 也是 $x$ 的函数)。
            
        - (3) 代入 $y'$ 并解出 $y''$。
            
    2. **参数方程 $x=x(t), y=y(t)$**：
        
        - (1) $y' = \frac{dy}{dx} = \frac{dy/dt}{dx/dt} = \frac{y'(t)}{x'(t)}$。
            
        - (2) 陷阱 ：$y'' = \frac{d^2y}{dx^2} = \frac{d}{dx} (y') = \frac{d/dt (y')}{dx/dt}$ (链式法则)。
            
        - (3) $y'' = \frac{d/dt \left( \frac{y'(t)}{x'(t)} \right)}{x'(t)} = \frac{\frac{y''(t)x'(t) - y'(t)x''(t)}{[x'(t)]^2}}{x'(t)}$。
            
        - (4) **公式**：$y'' = \frac{y''x' - y'x''}{(x')^3}$。
            

## VIII. 多元积分学：坐标系选择与对称性

**本章精神**：二重、三重积分 的计算瓶颈在于 (1) 坐标系选择，(2) 积分次序，(3) 对称性利用。

### A. 最佳坐标系的选择

- **汤/蒲精华**：
    
    1. **直角坐标**：积分区域是矩形、三角形（由直线围成）。
        
    2. **极坐标**：(a) 积分区域是圆、环、扇形。(b) 被积函数 $f(x,y)$ 包含 $x^2+y^2$ 或 $y/x$。
        
    3. **柱面坐标**：(a) 积分区域是柱体、锥体、旋转抛物面。(b) 被积函数包含 $x^2+y^2$。
        
    4. **球面坐标**：(a) 积分区域是球、锥（锥顶在原点）。(b) 被积函数包含 $x^2+y^2+z^2$ 或 $z / \sqrt{x^2+y^2}$ ( $\tan \phi$ )。
        
- **高级：广义坐标 (变量替换)** ：
    
    - **时机**：(1) 积分区域是“倾斜”的 (e.g., 顶点非 $(0,0)$ 的平行四边形)。(2) 被积函数是 $f(u(x,y), v(x,y))$ 的形式。
        
    - **示例**：求 $\iint_D (x-y)^2 \sin^2(x+y) dx dy$，$D$ 是 $(0,0), (\pi, -\pi), (2\pi, 0), (\pi, \pi)$ 围成的正方形。
        
    - **分析**：(1) $D$ 区域在 $xy$ 下是倾斜的。区域边界是 $x+y=0, x+y=2\pi, x-y=0, x-y=2\pi$。(2) 被积函数是 $(x-y)$ 和 $(x+y)$ 的组合。
        
    - 决策 ：使用 $u=x+y, v=x-y$ 变换。
        
    - **处理**：(1) $D$ 变为 $u,v$ 平面上的 $u \in [0, 2\pi], v \in [0, 2\pi]$。(2) 计算雅可比 (Jacobian) $J$。 $\frac{\partial(u,v)}{\partial(x,y)} = \det \begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix} = -2$。 $dx dy = |\frac{\partial(x,y)}{\partial(u,v)}| du dv = |1/(-2)| du dv = \frac{1}{2} du dv$。(3) $I = \int_0^{2\pi} \int_0^{2\pi} v^2 \sin^2 u \cdot \frac{1}{2} du dv = \frac{1}{2} (\int_0^{2\pi} v^2 dv) (\int_0^{2\pi} \sin^2 u du)$。(4) = $\frac{1}{2} [\frac{v^3}{3}]_0^{2\pi} [\frac{u}{2} - \frac{\sin 2u}{4}]_0^{2\pi} = \frac{1}{2} \cdot \frac{8\pi^3}{3} \cdot \pi = \frac{4\pi^4}{3}$。
        

### B. 轮换对称性

- **内容精要**： 明确提及。
    
- **二重积分**：如果积分区域 $D$ 关于 $y=x$ 对称 (e.g., $x^2+y^2 \le 1$ 或 $x+y \le 1, x\ge 0, y\ge 0$)。
    
    - **定理**：$\iint_D f(x,y) dx dy = \iint_D f(y,x) dx dy$。
        
    - **应用 1 (化简)**：$I = \iint_D \frac{f(x)}{f(x)+f(y)} dx dy$ ( $D$ 对 $y=x$ 对称)。
        
    - 处理：$I = \iint_D \frac{f(x)}{f(x)+f(y)} dx dy$。利用轮换对称性，$I = \iint_D \frac{f(y)}{f(y)+f(x)} dx dy$。
        
        $2I = \iint_D \frac{f(x)+f(y)}{f(x)+f(y)} dx dy = \iint_D 1 dx dy = \text{Area}(D)$。
        
        $I = \text{Area}(D) / 2$。
        
- **三重积分**：$\iiint_V f(x,y,z) dV$。
    
    - **时机**：如果 $V$ 具有 $x,y,z$ 轮换对称性 (e.g., $x+y+z \le 1, x,y,z \ge 0$ 或 $x^2+y^2+z^2 \le 1$)。
        
    - **定理**：$\iiint_V f(x,y,z) dV = \iiint_V f(y,z,x) dV = \iiint_V f(z,x,y) dV$。
        
    - 应用 2 (化简)：如果 $f$ 也是对称的 (e.g., $f=1$ 或 $f=x+y+z$)，则
        
        $\iiint_V x f(x,y,z) dV = \iiint_V y f(x,y,z) dV = \iiint_V z f(x,y,z) dV$。
        
    - **示例**：求 $\iiint_V x dV$，$V$ 是 $x+y+z \le 1, x,y,z \ge 0$ (四面体)。
        
    - 处理：(1) 设 $I_x = \iiint_V x dV, I_y = \iiint_V y dV, I_z = \iiint_V z dV$。(2) 由轮换对称性 ，$I_x = I_y = I_z$。(3) $I_x+I_y+I_z = 3 I_x = \iiint_V (x+y+z) dV$。(4) $I_x = \int_0^1 x dx \int_0^{1-x} dy \int_0^{1-x-y} dz = \int_0^1 x dx \int_0^{1-x} (1-x-y) dy$
        
        $= \int_0^1 x [\frac{1}{2}(1-x)^2] dx = \frac{1}{2} \int_0^1 (x - 2x^2 + x^3) dx = \frac{1}{2} [\frac{1}{2} - \frac{2}{3} + \frac{1}{4}] = \frac{1}{24}$。
        
    - (5) 此处 $3I_x = 3/24 = 1/8$。$\iiint_V (x+y+z) dV = 1/8$。
        
    - **思想提炼**：对称性 $I_x=I_y=I_z$ 在此是“验算”或“求解形心”的工具。形心 $\bar{x} = I_x / V$。$V = \int_0^1 \frac{1}{2}(1-x)^2 dx = 1/6$。$\bar{x} = (1/24) / (1/6) = 1/4$。由对称性，$\bar{x}=\bar{y}=\bar{z}=1/4$。
        

## 结论：拔高思想的精髓

本笔记的核心目的，是提供一套从“解题”到“解构”的思维框架。总结而言，高级数学分析的拔高路径并非记忆更多技巧，而是建立一个高效的“决策系统”：

1. **预处理 (Pre-processing)**：在计算之前，利用对称性、变换（如 $t=1/x$, $u=x+y$）或公式（如 Gauss, Stokes）对问题进行“净化”，消除病态项，简化积分域。
    
2. **结构识别 (Structure Identification)**：判断问题的“类型”。是 MVT 逆向工程？是级数的 Dirichlet 振荡？是高阶导数的 Leibniz 乘积？
    
3. **武器选择 (Optimal Weapon Selection)**：基于识别的结构，从“工具箱”中选择_最_适配的武器。例如，求 $f^{(n)}(0)$ 时，泰勒展开 往往优于 Leibniz。
    
4. **精密执行 (Precision Execution)**：在计算中（尤其是线面积分 ），严格规范符号、方向和边界，防止“计算失误”。
    

掌握这一系列“元分析”能力，即从“熟练套用公式”转变为“策略性地驾驭工具”，是从“汤家凤1800”到“蒲河平竞赛”的真正飞跃。
